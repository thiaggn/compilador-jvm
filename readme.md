# Meu Compilador

Este projeto implementa um compilador criado como trabalho da disciplina de Compiladores.  
Ele recebe programas escritos na linguagem X e gera **bytecode JVM**, permitindo execução na **Java Virtual Machine (JVM)**.

Funcionalidades principais:
- Análise léxica, sintática e semântica.
- Conversões implícitas de tipos para operadores lógicos e aritméticos.
- Redeclaração de variáveis com novo tipo em tempo de compilação.
- Geração de bytecode compatível com JVM.
